Welcome to YCweather.

YCweather is a software package designed to graph, organize, and share weather data throughout the Southwest Montana avalanche community. This program was developed by Andrew E. Slaughter at the Subzero Science and Engineering Research Facility (SSERF). For detailed information regarding YCweather please download the software documentation: help.pdf.

Installation

YCweather is a Windows based program and installs like any other application, however it requires MATLABâ€™s Component Runtime (MCR). First-time users of YCweather:

  • Download and install MCRinstaller.exe,
  • Download and install YCinstaller.exe, and
  • Run YCweather via YCweather.exe.
The first execution of YCweather may take several minutes, due to file decompression. Also, it is recommended that you install MCRinstaller using the default settings.

Historical Data

The following historical data sets are available (extract zip files into the database folder in the YCweather directory): 09-10, 08-09, 07-08, 06-07, and 05-06.
