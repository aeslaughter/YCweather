Welcome to YCweather.

YCweather is a software package designed to graph, organize, and share weather data throughout the Southwest Montana avalanche community. This program was developed by Andrew E. Slaughter at the Subzero Science and Engineering Research Facility (SSERF). For detailed information regarding YCweather please download the software documentation: help.pdf.

Installation

YCweather is a Windows based program and installs like any other application, however it requires MATLABâ€™s Component Runtime (MCR). First-time users of YCweather:

  • Download and install MCRinstaller.exe (it is recommend that you used the default settings),
  • Download and install setup.exe, and
  • Run YCweather via YCweather.exe from the installed location.
The first execution of YCweather may take several minutes, due to file decompression. Please note that YCweather, as discussed below, relies on dropbox for syncing current data. Therefore, by default if you have dropbox folder in the typical location, YCweather will automatically create a data directory inside this folder. Otherwise it will create a directory in your Documents folder or the installation directory, depending on your system. It is easy to move this database directory, details are provided in the user manual: help.pdf.

Historical Data

The following historical data sets are available (extract zip files into the database folder in the YCweather directory): 09–10, 08–09, 07–08, 06–07, and 05–06.

Current Data

The researchers at MSU maintain a current data set for YCweather using dropbox . If you would like access to this data please contact Andrew E Slaughter. Once you have been invited to the shared folder the following steps will setup YCweather to use this folder.
  • Download and install dropbox on your computer.
  • Start YCweather.
  • Select the Preferences option from the File menu in the YCweather application.
  • Change the database directory to the YCweather folder within your dropbox folder.
  • Make the changes the default by selecting the Set Default option in the Preferences window File menu. It is also possible to save the changes to a workspace for later recall, see the YCweather user manual for details: help.pdf
